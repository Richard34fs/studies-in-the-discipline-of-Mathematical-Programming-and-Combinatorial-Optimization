## Programação Linear:
    Um modelo de programação linear é uma respresentação de um problema de otimiaçãotimia
    Um modelo de programação linear é composto do seguinte:
        - Dados de entrada
        - Conjunto de variáveis - que podem assumir valoes reais
        - Conjunto de restições - que são equações ou inequações lineares
        - Função onjetivo - que é uma função linear
